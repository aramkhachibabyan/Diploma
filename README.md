# Գինու Արտադրության Օպտիմալացում

Խառը ամբողջաթիվ քառակուսային ծրագրավորման (MIQP) Java ծրագիր, որը օպտիմալացնում է ստանդարտ և պրեմիում գինիների արտադրությունը՝ մակսիմալացնելու շահույթը ռեսուրսների սահմանափակումների պայմաններում։

---

## Կախվածություններ (pom.xml)

```xml
<!-- Օպտիմալացման գրադարան -->
<dependency>
    <groupId>org.ojalgo</groupId>
    <artifactId>ojalgo</artifactId>
    <version>55.0.1</version>
</dependency>

<!-- Գծապատկերների գրադարան -->
<dependency>
    <groupId>org.jfree</groupId>
    <artifactId>jfreechart</artifactId>
    <version>1.5.4</version>
</dependency>
```

## Կառուցում և Գործարկում

```bash
# Կառուցում
mvn clean package

# Գործարկում
java -Dfile.encoding=UTF-8 -jar target/wine-opt.jar
```

> **Նշում:** `-Dfile.encoding=UTF-8` պարամետրը անհրաժեշտ է հայատառ տեքստը ճիշտ ցուցադրելու համար։

## Մաթեմատիկական Մոդել

### Նպատակային ֆունկցիա

```
max  Σ [ A[i]·X[i] − B[i]·X[i]² − C[i]·X[i] ]  −  Σ F[j]·Y[j]
      i                                               j
```

| Նշան | Նկարագրություն |
|------|----------------|
| `X[i]` | i-րդ գինու արտադրության քանակ (ամբողջ թիվ, ≥ 0) |
| `Y[j]` | j-րդ պրեմիում գինու երկուական փոփոխական (0 կամ 1) |
| `A[i]` | Գծային եկամտի գործակից |
| `B[i]` | Շուկայական հագեցվածության գործակից |
| `C[i]` | Փոփոխական ծախսի գործակից |
| `F[j]` | Պրեմիում գինու ֆիքսված ծախս |

### Սահմանափակումներ

**Ռեսուրսների սահմանափակումներ:**
```
Σ r[k][i] · X[i]  ≤  R[k]     ∀ k
  i
```

**Big-M սահմանափակումներ (պրեմիում գինիների համար):**
```
X[nStandard + j]  ≤  M · Y[j]     ∀ j
```

## Մուտքագրվող Տվյալներ

Ծրագիրը հաջորդաբար խնդրում է հետևյալ տվյալները.

| # | Տվյալ | Նկարագրություն |
|---|-------|----------------|
| 1 | Ստանդարտ գինիների քանակ | `nStandard` |
| 2 | Պրեմիում գինիների քանակ | `nPremium` |
| 3 | A գործակիցներ | Եկամտի գծային գործակիցներ բոլոր գինիների համար |
| 4 | B գործակիցներ | Քառակուսային գործակիցներ (հագեցվածություն) |
| 5 | C գործակիցներ | Փոփոխական ծախսի գործակիցներ |
| 6 | F գործակիցներ | Ֆիքսված ծախսեր **միայն** պրեմիում գինիների համար |
| 7 | Ռեսուրսների քանակ | `nResources` |
| 8 | R[k] արժեքներ | Յուրաքանչյուր ռեսուրսի ընդհանուր հասանելիություն |
| 9 | r[k][i] արժեքներ | Ռեսուրսների սպառման մատրիցա |
| 10 | M հաստատուն | Big-M հաստատուն պրեմիում գինիների սահմանափակման համար |

## Ելքային Արդյունքներ (Console)

Ծրագիրը տպում է.

- **Արտադրության քանակներ** — յուրաքանչյուր գինու համար `X[i]`, եկամուտ, փոփ. ծախս, շահույթ
- **Պրեմիում գինիների որոշումներ** — `Y[j]` արժեք, արտադրման կարգավիճակ, ֆիքսված ծախս
- **Շահույթի կառուցվածք** — ընդհանուր եկամուտ, ծախսեր, շահույթ
- **Ռեսուրսների օգտագործում** — յուրաքանչյուր ռեսուրսի օգտ. / մնացած / տոկոս

│
├── main()                        — Մուտք, մոդել, լուծում, console ելք

├── drawResourceStackedByWine()      — Գծ. 4: ռեսուրսնեի ծախս
│
├── styleMainTable()              — Օժ.: գլխավոր աղ. ոճ
├── styleChart()                  — Օժ.: գծ. ոճ
├── styleCategoryAxes()           — Օժ.: առանցքների ոճ
├── styleXYAxes()                 — Օժ.: XY-առանցքներ
